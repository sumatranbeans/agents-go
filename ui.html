Agents: Go

        
Autonomous agent scheduler for Claude Code

        
Schedule and run Claude Code agents across multiple projects. They operate on autopilot — each agent reads its own project context, executes its tasks, and exits cleanly.

        

          http://localhost:3847
          
        

      


      
      

        

          
        

        
Add Projects

        
This scheduler manages multiple projects simultaneously. Each project points to a code directory and gets its own independent set of agents and schedules.

        
Click + Add Project on the Dashboard to get started. You can add as many projects as you need.

      


      
      

        

          
        

        
Add Agents

        
Create agents within each project. Give them a name and an invocation prompt — the instruction they receive when summoned.

        
    
          
  • Agents read their own project and persona files to understand their role and act accordingly
    • 
          
  • Every session is completely fresh — no state or memory carries over between runs
    • 
          
  • Set how often each agent runs: which hours, which days, and at what interval
    • 
        

      


      
      

        

          
        

        
Set Schedule

        
Configure exactly when each agent runs with three controls:

        
    
          
  • Hours — Toggle any combination of 0–23 in your local time
    • 
          
  • Days — Select which days of the week, Monday through Sunday
    • 
          
  • Interval — Every 15, 30, 45, or 60 minutes within each active hour
    • 
        

      


      
      

        

          
        

        
Watchouts

        

          Token Usage — Each agent session consumes API tokens. An agent at 15-min intervals across 8 hours burns 32 sessions per night. Start conservative with 60-min intervals and monitor your Anthropic usage dashboard.


          Memory & Storage — No state persists between agent sessions. Sprint logs are kept for 14 days, then automatically cleaned up.


          First Run — Claude Code will show a one-time permissions bypass warning. Select "Yes, I accept" — it remembers permanently.


          Stuck Agents — Use the "Kill All Running" button on the Dashboard to terminate any hung sessions.
